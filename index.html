Importar datos desde SAP

Para pendientes SAP puedes importar Excel/CSV con columnas como Fecha plan. de carga, REF REAL y Nº de cargas. Para Cargas previstas de la semana únicamente se utilizará la columna REF REAL; si REF REAL está vacío, la fila no se importará como carga prevista. Las referencias de envasado se pasarán a “Envasados pendientes por día”, salvo si contienen FV016 o FC. En “Referencia envasado” se mostrarán juntas las columnas PRODUCTO, REF REAL y REF SAP cuando existan.

Importa los pendientes SAP para actualizar las cargas previstas de la semana entre las dos fechas indicadas.
